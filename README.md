This project implements a Handwritten Digit Recognition System using Python and Machine Learning. The model is trained on the MNIST dataset and can accurately recognize handwritten digits from 0 to 9.

Technologies Used
Python
NumPy
Pandas
Matplotlib
Scikit-learn
Joblib
Model
MLP (Multi-Layer Perceptron) Classifier
Hidden Layers: (128, 64)
Data Normalization
Train-Test Split for Evaluation
Features
Digit classification (0–9)
Data preprocessing and normalization
Model training and evaluation
Saved trained model for future predictions
Dataset
MNIST Handwritten Digits Dataset
